## 📊 Project Overview / Descripción del Proyecto

This project is an interactive web dashboard designed to perform exploratory data analysis (EDA) on vehicle listings in the United States.

Este proyecto es un dashboard interactivo que permite realizar análisis exploratorio de datos (EDA) sobre anuncios de vehículos en Estados Unidos.

The application enables users to explore patterns in pricing, mileage, and vehicle characteristics through dynamic visualizations.

La aplicación permite explorar patrones en precios, kilometraje y características de los vehículos mediante visualizaciones dinámicas.

---

## 💼 Business Context / Contexto de Negocio

Understanding the vehicle market is essential for identifying pricing trends and detecting anomalies in listings.

Comprender el mercado de vehículos es clave para identificar tendencias de precios y detectar anomalías en los datos.

Este tipo de análisis puede apoyar:

## ⚙️ Features / Funcionalidades

- Interactive filters (price, year, vehicle type)  
- Dynamic histograms for mileage distribution  
- Scatter plots for price vs mileage analysis  
- Real-time data exploration  
- Key metrics (KPIs) display
